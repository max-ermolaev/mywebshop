MyWebShop is a VS2010 project implemented for obtaining ASP.NET and C# experience.


Что я использовал:
- ASP.NET Web Forms
- Entity Data Framework
- MS Sql Server 2008
- систему версионной миграции баз данных [Wizardby][]
- бесплатный шаблон для дизайна
- информацию о продуктах из базы данных [TailspinSpyworks][]

    [Wizardby]: http://code.google.com/p/octalforty-wizardby/
    [TailspinSpyworks]: http://www.asp.net/web-forms/samples/tailspin-spyworks

Можно:
- просматривать товары по категориям
- осуществлять поиск товара по наименованию
- операции с корзиной: добавить, удалить, изменить количество
- делать заказы
